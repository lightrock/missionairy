# Completion Report Failure Example

## Report

I fixed the PR template and workflow. Everything should work now.

## Why this fails the rubric

This report collapses patch, expectation, and verification.

It does not say:

- which files changed
- whether a workorder was used
- which `Checks run`
- which `Checks not run`
- whether a lesson learned was needed
- what remains uncertain
- what the `Verification status` is

It may be true that the patch is correct, but the report does not provide verification evidence.
